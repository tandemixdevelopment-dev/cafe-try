# ☕ Coffee Haven - Сайт кафе на React

Современный и красивый сайт кафе, построенный на React с использованием Vite и Tailwind CSS.

## 🚀 Возможности

- 📱 Полностью адаптивный дизайн (мобильный, планшет, десктоп)
- ☕ Красивая страница меню с категориями
- 🖼️ Галерея кафе
- 📞 Форма контактов
- 🎨 Современный дизайн с кофейной цветовой гаммой
- ⚡ Быстрая загрузка благодаря Vite

## 🛠️ Установка и запуск

### Требования
- Node.js 16+ 
- npm или yarn

### Шаги установки

1. Перейдите в директорию проекта:
```bash
cd cafe-try
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте браузер и перейдите на `http://localhost:3000`

## 🏗️ Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## 📄 Страницы сайта

- **Главная** (`/`) - Приветственная страница с информацией о кафе
- **Меню** (`/menu`) - Полное меню с кофе, десертами и напитками
- **О нас** (`/about`) - История кафе и информация о команде
- **Галерея** (`/gallery`) - Фотографии интерьера и блюд
- **Контакты** (`/contact`) - Контактная информация и форма обратной связи

## 🎨 Цветовая схема

- **Кофе**: `#6F4E37` - основной темный цвет
- **Крем**: `#F5E6D3` - светлый фоновый цвет
- **Акцент**: `#D4A574` - золотистый цвет для акцентов

## 📦 Зависимости

- **React 18.2** - библиотека для построения пользовательского интерфейса
- **React Router 6.8** - маршрутизация между страницами
- **Vite 4.3** - быстрый инструмент сборки
- **Tailwind CSS 3.3** - утилиты CSS для быстрого стилирования
- **PostCSS 8.4** - обработка CSS
